# ATA-IT-Website

## Run locally

```bash
# clone this repository
git clone https://github.com/MetasPattaraamornpan/ATA-IT-Website.git

# build docker image
docker build -t [NAME] .

# run docker image
docker run -d -p [PORT]:5000 [NAME]

# install lib
npm install

# build webpack
npm run build

# run development mode
npm run dev

# run the app
npm start

# Web path
http://localhost:5000 

```
